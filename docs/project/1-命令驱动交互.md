# 命令驱动交互

## 状态图

```
                    ┌─────────────────┐
                    │    Home Page    │
                    │  (CommandCenter)│
                    └───┬──┬──┬──┬───┘
                        │  │  │  │
               ┌────────┘  │  │  └────────┐
               ▼           ▼  ▼           ▼
        ┌──────────┐ ┌──────────┐ ┌──────────┐
        │ /brain   │ │ /role    │ │ /chat    │
        │ BrainModal│ │RoleModal │ │ ChatView │
        │ (弹窗)   │ │(弹窗+头像)│ │ (页面)   │
        └──────────┘ └──────────┘ └──────────┘
                    ESC返回Home
```

三种命令，三种呈现形态：
- `/brain` / `/role` → 模态弹窗，覆盖在首页之上
- `/chat` → 页面模式切换，Home 隐藏，Chat 展示

---

## 命令表

| 命令 | 功能 |
|------|------|
| `/brain` | 管理大脑配置（增删改查、测试连接） |
| `/role` | 管理人物（创建/编辑，绑定大脑，选择头像） |
| `/chat` | 进入聊天模式 |
| `/image` | 进入文生图模式 |

## 首页布局

```
┌──────────────────────────────────────┐
│ ☰                     OpenRise    │ ← fixed top-0 h-12
│                                      │
│         ZZZZZZZZZZZZZZ               │ ← 炭笔大标题
│         OOOOOOOOOOOO                 │    3层偏移叠加
│         EEEEEEEEEEEE                 │    wiggle动画
│                                      │
│            > 输入 / 开始...           │ ← 命令输入框
│            ─────────────────          │    shaky-line
│                                      │
└──────────────────────────────────────┘
```

- OpenRise 标题 translateY(-40px)，略高于垂直中心
- 命令输入框 `max-w-lg` 居中，`>` 前缀
- 左上 `☰` 打开侧边栏，右上 OpenRise 为返回按钮（chat 模式时）

---

## 页面骨架（PageShell）

`mode` 状态控制全部页面切换：`'home' | 'chat' | 'image'`

- 顶部栏：所有 mode 共享，`h-12 flex items-center px-10`
  - chat 模式显示角色名称（AvatarIcon + 文字），absolute 居中
  - 侧边栏打开时名称右移 128px（transition-transform）
- 侧边栏：flex shrink-0，`width: 256px ↔ 0` transition
  - chat 模式显示人物列表，点击切换对话
  - image 模式显示可文生图的人物
- 主区域：根据 `mode` 渲染对应组件
  - home → CommandCenter
  - chat → ChatView
  - image → ImageView
