## 状态图

```
                    ┌─────────────────┐
                    │    Home Page    │
                    │  (CommandCenter)│
                    └───┬──┬──┬──┬──┬──┘
                        │  │  │  │  │
               ┌────────┘  │  │  │  └──────────┐
               ▼           ▼  ▼  ▼             ▼
        ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
        │ /brain   │ │ /role    │ │ /chat    │ │ /agent   │
        │ BrainModal│ │RoleModal │ │ ChatView │ │ AgentView│
        │ (弹窗)   │ │(弹窗+头像)│ │ (页面)   │ │ (页面)   │
        └──────────┘ └──────────┘ └──────────┘ └──────────┘
                    ESC返回Home
```

四种命令，三种呈现形态：
- `/brain` / `/role` → 模态弹窗，覆盖在首页之上
- `/chat` → 页面模式切换，Home 隐藏，Chat 展示
- `/agent` → 页面模式切换，Home 隐藏，Agent ReAct 界面展示

## 命令表

| 命令 | 功能 |
|------|------|
| `/brain` | 管理大脑配置（增删改查、测试连接） |
| `/role` | 管理人物（创建/编辑，绑定大脑，选择头像） |
| `/chat` | 进入聊天模式 |
| `/agent` | 进入 Agent 模式（ReAct 任务执行） |

## 页面骨架（PageShell）

`mode` 状态控制全部页面切换：`'home' | 'chat' | 'agent'`

- 顶部栏：所有 mode 共享，`h-12 flex items-center px-10`
  - chat/agent 模式显示角色名称（AvatarIcon + 文字），absolute 居中
  - agent 模式额外显示 ✕ 切换角色按钮
  - 侧边栏打开时名称右移 128px（transition-transform）
- 侧边栏：flex shrink-0，`width: 256px ↔ 0` transition
  - chat 模式显示人物列表，点击切换对话
  - agent 模式显示会话列表（AgentSessionList），含新建/删除/选择操作
- 主区域：根据 `mode` 渲染对应组件
  - home → CommandCenter
  - chat → ChatView
  - agent → AgentView

## Agent 模式

Agent 模式 (`/agent`) 启动一个 ReAct 任务执行界面。用户输入任务描述后，后端 Agent 循环会调用工具（读/写/编辑文件）来完成任务。

### 会话管理

- 侧边栏（点击左上 ☰）显示 Agent 会话列表
- 「+ 新建会话」创建新会话，首条消息自动生成标题
- 切换会话时从数据库加载历史消息

### 内置命令

Agent 输入框中支持以下斜杠命令（与普通任务消息同一输入框，通过前缀 `/` 区分）：

| 命令 | 功能 |
|------|------|
| `/trust` | 查看当前信任路径列表 |
| `/trust add <path>` | 添加一个信任路径 |
| `/tool` | 查看 Agent 可用工具（read_file、write_file、edit_file） |
| `/help` | 显示命令帮助 |

命令结果以对话消息形式展示（用户消息 + 助手回复），融入聊天流。

### Tab 自动补全

输入 `/` 开头命令时，按 Tab 自动补全到匹配的命令前缀。例如 `/tr` → Tab → `/trust `。补全后缀在输入框中以浅色文字预览。
